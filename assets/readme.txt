Connect Wordpress with x2CRM Contact Relationship and Marketing Automation. Requires Contact Form 7

== Description ==
Collect Contact Form 7 submissions to your x2CRM installation using x2CRM's Legacy API. Form submissions from your WordPress website will submit to x2CRM as Contact types. All the predefined Contact fields have been included for use in your forms. Fields should be added to your Contact Form 7 form using the following as form name: firstName, lastName, email, phone, phone2, visibility, leadtype, leadSource, backgroundInfo, website, address, address2, city, state, zipcode, country, leadscore.  



== Installation ==
1. Upload WP-Revive files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' screen in WordPress

3. Use the Settings->x2CRM Options screen to configure the plugin

== Frequently Asked Questions ==
= Does this plugin work with newest WP version and also older versions? =

Yes, this plugin works really fine with WordPress 4.7.3.
It also works with WordPress 4 or greater.

= Do I have to use Contact Form 7?=

Yes, the plugin was developed to interact with Contact Form 7
